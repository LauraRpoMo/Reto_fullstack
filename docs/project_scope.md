# Alcance del Proyecto

## Nombre del proyecto
Tienda Online Fullstack

## Objetivo
Desarrollar una aplicación web fullstack para una tienda online que consuma una API REST, ofreciendo una experiencia moderna, intuitiva y responsiva para usuarios en dispositivos móviles y desktop.

## Público objetivo
Personas interesadas en comprar productos en línea de forma rápida y sencilla.

## Funcionalidades principales
- Registro e inicio de sesión de usuarios
- Galería dinámica de productos
- Búsqueda en tiempo real
- Paginación de productos
- Carrito de compras con cantidades y total
- Vista previa del checkout
- Persistencia local o integración futura con Firebase

## Valor diferencial
Diseño moderno inspirado en la plantilla Kaira, priorizando accesibilidad, experiencia de usuario y arquitectura escalable en React.
