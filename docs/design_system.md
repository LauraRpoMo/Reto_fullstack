# Sistema de Diseño

## Inspiración visual
La aplicación toma como referencia la plantilla Kaira, destacando un estilo minimalista, elegante y centrado en producto.

## Paleta de colores
- Fondo principal: Blanco (#FFFFFF)
- Texto principal: Negro suave (#1A1A1A)
- Color acento: Beige cálido (#D9CBB6)
- Botones: Negro (#000000)
- Texto secundario: Gris medio (#6B7280)

## Tipografía
- Principal: Sans-serif moderna
- Jerarquía clara con tamaños amplios para títulos

## Estilo visual
- Diseño limpio y espacioso
- Uso de tarjetas para productos
- Bordes suaves y componentes reutilizables
- Experiencia responsive mobile-first
